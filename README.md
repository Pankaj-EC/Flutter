# Solar Streetlight Tracker

A Flutter application designed to monitor and manage solar streetlights. This project aims to provide a user-friendly interface for tracking device status, monitoring energy consumption, and receiving real-time updates on device performance.

## Features

* Device List Screen: Displays a list of all registered solar streetlights, including their ID, creation time, and status.
* Device Detail Screen: Provides detailed information about a selected device, including its status, energy consumption, and historical data.
* Real-time Updates: Receives live updates on device status and energy consumption.
* Load More Data: Allows users to load more historical data for a device.
* Update Device Status: Enables users to remotely switch devices on or off.

## Technologies Used

* Flutter for the mobile application
* Dart for the programming language
* HTTP for API communication
* JSON for data exchange

## Setup

1. Clone the repository.
2. Install Flutter and Dart on your machine.
3. Set up your development environment.
4. Run the application on a simulator or physical device.
